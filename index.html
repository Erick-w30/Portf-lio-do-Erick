Corrida de 4 Jogadores

Jogo de Corrida para 4 Jogadores em p5.js Este é um jogo simples de corrida criado no editor p5.js, voltado para 4 jogadores. Cada jogador controla um carro que se move horizontalmente da esquerda para a direita na tela. O objetivo é ser o primeiro a alcançar a linha de chegada do outro lado. Os controles são distribuídos no teclado da seguinte forma:

Jogador 1: tecla A

Jogador 2: tecla S

Jogador 3: tecla D

Jogador 4: tecla F

Cada vez que um jogador pressiona sua tecla, seu carro avança uma pequena distância. O jogo continua até que um dos carros cruze a linha final, sendo declarado o vencedor. É um jogo competitivo, rápido e divertido, ideal para jogar com amigos no mesmo teclado!

Jogo Pong

Jogo Pong para 2 Jogadores em p5.js Este é um clássico jogo Pong recriado no editor p5.js para dois jogadores. O objetivo é simples: rebater a bola usando as raquetes e impedir que ela passe por seu lado da tela. O jogo termina quando um dos jogadores atinge uma pontuação definida.

Jogador 1:

W: mover a raquete para cima

S: mover a raquete para baixo

Jogador 2:

Seta ↑: mover a raquete para cima

Seta ↓: mover a raquete para baixo

A bola se move automaticamente e rebate nas bordas superior e inferior. Se um jogador não conseguir rebater a bola, o adversário marca ponto. É um jogo competitivo, rápido e ótimo para dois jogadores no mesmo teclado.

Efeito Monalisa

Jogo com Efeito Monalisa no p5.js Este é um projeto divertido feito no editor p5.js que utiliza o famoso "efeito Monalisa", no qual os olhos de um personagem seguem o cursor do mouse, dando a impressão de que ele está sempre te observando. O personagem aparece no centro da tela com dois olhos grandes e expressivos. Conforme o jogador move o mouse, as pupilas se ajustam e se movem na direção do cursor, simulando o efeito de olhar. Apesar de simples, esse recurso visual cria uma interação divertida e curiosa entre o jogador e o personagem. Esse projeto é ideal para aprender sobre coordenadas, mapeamento de posições e interatividade no p5.js.
